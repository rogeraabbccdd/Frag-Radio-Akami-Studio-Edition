# Frag-Radio-Kento-Edition
Improved version of [FragRadio](http://fragradio.com/) sourcemod plugin

## Changes In This Edition
* Colors Support.
* Translation Support.
* Auto Play Radio.
* Random Advert.
* Auto Generate Config at cfg/sourcemod/plugin.kento_fragradio.cfg
